## 개발 환경

- Unreal Engine 5.7
- Visual Studio 2022 / C++
- Enhanced Input
- Git + Git LFS

## 구현 내용

- Enhanced Input 기반 3인칭 캐릭터 이동 및 시야 조작
- 인터페이스(`IInteractable`) 기반 상호작용 구조
- 구체 오버랩을 통한 상호작용 대상 감지
- 문 / 상자 액터 및 재화 획득
